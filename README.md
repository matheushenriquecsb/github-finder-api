## Installation

### 1. Clone o repositório

```bash
git clone https://github.com/matheushenriquecsb/githubfinder-api
```

### 2. Acesse o diretório

```bash
cd githubfinder-api
``` 

### 3. Instalando dependências

```bash
npm install
``` 

### 4. Run the application

```bash
npm run dev
```
